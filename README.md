# Blog

写点什么吧，在我决定把这个业余项目终结之前。

2021年末，我还是没有搞清楚 GitHub Action、Jekyll 和 GitHub Pages 的关系和工作原理，我看到一个外国人的博客，觉得不错，就一点点模仿，加入自己觉得不错的元素，框架，最后就成了现在这个样子。

数据源拉取也是照着别人的 GitHub Action 脚本一点点改的，思路是听播客的时候被提点到的，觉得很有意思，就想试试，结果还真能用。

但是做的时候没想太多，我其实并不喜欢在 ISSUE 里面写博客，我在一篇文章里面也总结过这个问题，所以用它的频率不再，项目完全陷入了停滞状态。

现在，这个项目存在的意义，就是减少互联网上的死链接，还给读者一个仍然可持续访问的博客，未来我的博客不再在这个仓库更新，但依然会用脚本把文章推送到这里。

谢谢你能看到这里。
