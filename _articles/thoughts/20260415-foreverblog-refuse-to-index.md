上个月 重构 Jekyll 博客 之后，信心满满地去投稿十年之约，然后今天早上终于得到了回复：

![](https://pub-89c11651a8434f18a530bd6f93e399da.r2.dev/2026/20260415210049927.webp)

最终决定是被拒绝啦，可以说毫不意外吗？因为点投递按钮的时候心里就有点「侥幸心理」，因为我的博客里确实存在一些对劳动法和当今社会体制不作为的吐槽。

就结果而言，我的「感觉」没有错。但这始终是一种感觉，没有白纸黑字的确定感，悬浮在空中，让人捉摸不定。我们来看看十年公约：

## 十年之约

我已经不记得从哪里了解到这个组织的了，但我觉得十年这个期限是极好的，因为一个域名的续费最长的时间，就是十年，无论什么时候，你只能保证未来十年你的域名可用。

我的这个域名 bgzo.cc 大概是从 21 年起开始注册的，截止目前已经有 5 个年头了，按十年期限算的话，我已经完成了一半的进度。

但我一个人说的不作数，对吧，我们来看看十年之约的第二条：

> 1. 您的博客正在使用二级域名，您必须拥有该二级域名对应主域名的所有权，不接受三级及以下级别域名。
> 2. 备案要求：加入『十年之约』的博客不限制是否备案。如果您的博客已备案，需保证您的备案信息为您个人的真实信息，且域名所有人与备案信息一致，并按工信部和公安机关的要求在博客页脚标注备案号。
> 3. 内容要求：
> 	1. 生活记录类：包括但不限于（旅行、摄影、自传、文学、历史、日常记录）等。
> 	2. 加入『十年之约』的生活记录类博客，原则上需已建立十二个月及以上，累计发布文章不少于 15 篇，其中原创文章占比不低于 75%。不得发布任何违反中华人民共和国现行法律法规的内容。
> 	3. 学习记录类: 鉴于实现步骤的大体相似，因此坚决杜绝雷同，建议记录实现过程中各种意外的 debug,文中可以轻易搜索到的实现步骤尽量引用原始博文，但严禁大篇幅搬运，少数被删库的稀缺资源不在此条约束范围之内。 加入『十年之约』的学习记录类博客，原则上需已建立十二个月及以上，累计发布文章不少于 30 篇，其中原创文章占比不低于 70%。不得发布任何违反中华人民共和国现行法律法规的内容。
> 4. 优秀文章展示：凡加入『十年之约』的用户，可在加入满十二个月及以上时向项目组申请把站点加入『十年之约』「RSS」订阅地址：（[https://www.foreverblog.cn/feeds.html](https://www.foreverblog.cn/feeds.html)）。出现滥用行为会导致您的站点「RSS」被移除,原因包括但不限于频繁出现无意义水文、违规文章等。
> 5. 博客内容版权：加入『十年之约』的博客,博主必须对其发布的全部原创内容拥有完整、独立的版权。凡转载或引用他人作品,博主务必确保按照原作者的版权要求或留下出处。博客所使用的程序、主题、框架等,需在页脚或站内显著位置进行明确的版权声明。若博客因版权问题引发纠纷,我们将依据相关约定,对该博客按照违约进行处理。
> 6. 不允许加入『十年之约』的博客：
> 	1. 宣扬或⿎吹暴⼒、⾊情、犯罪、赌博、恐怖主义、邪教、纳粹、战争、种族歧视等违反法律、道德、引导对立、⼈道主义的博客；
> 	2. 通过捏造事实、断章取义、片面报道或引用无公信力材料等手段，蓄意误导公众对特定国家/地区、组织或个人的认知的博客；
> 	3. 侵犯他⼈知识产权、版权的博客；
> 	4. 发布或泄露他人隐私信息的博客。
>
> via: https://www.foreverblog.cn/treaty.html

我一条条过：

1. 二级域名 blog.bgzo.cc
2. 本博客无备案
3. 内容较杂：技术（30%）+ 个人（70%），可能有一些大陆媒体无法过审的文章；
4. 本质不算要求，无无意义水文
5. 内容全部原创
6. 无过激内容，无误导认知博客，无侵权，泄漏隐私内容；

虽然自我感觉良好，但最终只换来一句：

> 您的站点目前暂不符合《十年公约》中所列之内容。

没有更进一步细节，甚至连这份邮件我都怀疑是模版，因为我根本找不到邮件随附的《十年之约》，上面的条款还是我从网站上重新扒下来的。

怎么说呢？这味道我可太熟悉了，这就是审查的魅力，你永远不知道哪篇文章、哪个段落，哪句话不符合对方眼里「标准」，直到你把想说的话咽回肚子，不再创作。

## 要改变吗

我不知道。

在刚收到邮件的那一个小时，我还构想过要不把博客圆和这里的博客完全做统一，把一些 #politics 和 #flamewar 的内容过滤掉，这样也许受众面更广。

甚至，我还有一些灾难化的想象：因为「十年之约」经营了这么多年，其实也算一个值得参考的审查单位？如果我的这些文字未能通过审查，那会不会也能作为未来被定罪的标本？我会不会就此把我最珍贵的十几年交代进去？

我更迷茫了。

最后，我完全放弃了这些想法，一方面，我已经没有再审查我过去写的那些想法的经历了，二来，因为我在博客圆额外同步了 How to 的文章过去，严格意义上来说，两边是完全不同的存在形式，如果再调整一遍，又是一项巨大的改动。

我不想折腾了，我想写点东西。

而且，你看到的这些文字，其实已经是经过我两次审查的结果了。我已经把大部分有争议的内容已经过滤掉了：

```python
"""
Global Variables:
"""
SENSITIVE_TAGS = {
    'flamewar',         # 争议相关
    'love/letter',      # 私情相关
    'sex',              # 性相关
    'politics',         # 政治相关
    'religion',         # 宗教相关
    'pointless',        # 无意义内容
    'spam',             # 垃圾内容
    'private',          # 私人内容
    'relationship',     # 关系相关
    'privacy',          # 隐私相关
    'nsfw',             # NSFW
}
```

都已经发到大陆之外了，至少我希望自己写的东西可以自由一点，就像我在博客圆里面写的 https://www.cnblogs.com/bgzo/p/18956083 ，我不会再因此改变什么。

虽然没能通过审核，但能成为未通过的一份子也足够幸运。
