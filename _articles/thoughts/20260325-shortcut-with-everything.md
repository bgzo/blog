我一直认为，人生中的每件事情都是有捷径的。

比如数学有二级公式，做题有答题模板，在一切都可以理性分析的前提下，一定存在比直来直去更加取巧的办法。

就拿数学的二级公式举例，两位身心素质完全相同的孩子，唯一区别就是是否知道一些应试教育的二级公式，他们同时去解一道高中的圆锥曲线的题，不会的孩子往往得靠草稿纸手算七八分钟，还不一定准，知道二级公式的孩子只需要带入参数算一下，一分钟内就出来了，这在应试教育里是致命的差距。

在这样一个功利的社会中，很多事情都是这个样子，知识学生时代这个捷径可能是一位网课老师，一堆教材辅助，如今的大模型时代，他甚至可能是 LLM 随机生成的一段话。

自然的，一个很不舒服的事实是，不是任何人都能有运气在正确的时间，正确的找到一条适合自己的捷径，尤其在学校这种一节跟着一节，填鸭式的教育场景，学生的时间有限，这些不那么幸运的孩子怎么办？

更难的是，一次小小的错误会在未来无数次重演，带来的不再是连点成线，连线成面的成就感，而是一次次对过去自己的否定，挫败，最终就会走向焦虑、内耗、摆烂。认为自己不适合这一切。

事实就是这样，我也一样，我对自身的 CS 水平就是上述的状态，已经毕业三年的我，每天回到家已经不想再看电脑了，更别提大学讲的那些专业知识。最难过的是，这些专业课的专业词汇，还会在我脑袋里打作一团，阻碍我学习别的东西。我依然搞不懂那些最艰深的算法问题，红黑树，动态规划等等，即使看一眼题解就能想起来，但现在我就是想不出来，我还是会下意识去，模拟，遍历，循环，就像我日常在写的 Java 一样，而且就算写 Java ，日常也用不到 GC 调优，内存分析等等。

如今想想，真不如让 LLM 教我，每个月期 4000 块的 TOKEN（一个月 1000），我还学不会那些专业知识？

可 LLM 不是银弹，LLM 普及的当下，我还应该继续学那些算法，背诵面试八股文吗？

我不知道，所以最近我整日整夜焦虑地睡不踏实，我再也无法锚定一个方向去努力了，锚点时时刻刻在变化！这就是不确定性。

好难～今晚我应该做什么呢？养虾，学英语，刷题，还是早点睡觉？

早点睡吧。
