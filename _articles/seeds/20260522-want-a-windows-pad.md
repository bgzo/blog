好想要一个 Windows 平板，但是：

> 二合一平板看着很美好，实际用起来就是一坨 Shit ，散热散热不行续航也不行，糟糕的键盘手感，更不用说整体重量也没啥优势。Surface 的品控是一个非常严重的问题，强烈建议购买传统形态的 Windows 轻薄本即可。其实幻 13 也不错，或者新一代 CPU 马上就要铺货上市了，你的预算又都是高端路线很快就有相关产品可以购买了
> https://www.v2ex.com/t/1053587

其实还是有一个候选者的，联想的 Yoga 可以 360 翻折，于是我们可以到一个 2 斤重的平板，它跑的是 Windows，我觉得还是可以胜任一些简单的临时工作、娱乐。

可惜太贵了，1W 呢，如果 1/3 的价钱我一定会入手。而且它可是灌黑胶的联想，所以更不想买了。
