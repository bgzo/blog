看到 V 的帖子: https://www.v2ex.com/t/1210050, 说 https://my.dnshe.com/index.php 支持永久域名，一开始看不上，最终想试试，才发现老费劲了：

1. [首页](https://www.dnshe.com/) 提供了一些域名的选择，`.de5.net`/ `.us.ci`，进去其实是没货的，取而代之的却是 `cc.cd` / `cn.mt`/`ccwu.cc`/`bbroot.com` 这几个；
2. 没有限制二级域名的长度，但是真的进去注册的时候才会知道其实是有最少 3 位的限制；
3. 很难注册，能想到的全部都被注册了：
	1. abc
	2. web
	3. app
	4. api
	5. dev
	6. vvv
	7. blog
	8. wiki
	9. api
	10. 404
	11. ssh
	12. img
	13. 禁止一切产品名字，比如 deepseek、bing、google 等等

最终只注册到这几个，不想搞了，还得拉人助力，一个域名拉 5 个人，一个人 10 个名额，所以保守起见一个人最多只能拿两个域名：

- exe.cn.mt
- nix.cn.mt
