最近把几乎所有的仓库都转移到 bgzo-sandbox 里面了，这个组织其实就是我的沙箱，其实我一直都想整个小号的，但是 GitHub 一直不让整小号，要不然大小号全给你撸了。

> 一个人或一个法律实体最多可以拥有一个免费帐户（如果您选择同时控制一个机器帐户，可以，但是它只能用于运行机器）。
> https://docs.github.com/zh/site-policy/github-terms/github-terms-of-service#3-account-requirements

- https://www.v2ex.com/t/836086
- https://www.v2ex.com/t/817831

我是参考的 [@walfie](https://github.com/walfie)，他就有这么个组织 https://github.com/walfie-sandbox ，名字我都是抄人家的，主要就是把自己一时兴起的玩具、存档的、机器人（GitHub Action）运维、Fork 的项目，放在沙箱里面，这个还是有蛮多需求的，上面被封、开小号的人也都是这么个逻辑，但我一直无法理解的是：

> 将一个帐号在其他开源项目的贡献、PR 、Issue 、讨论全部删除不留痕迹（普通注销的帐号都会变 Ghost ，而 ban 的帐号什么都不留） ，这是对开源贡献者最大的侮辱，也是对开源社区的侮辱。完全不把个人的无私付出当作一回事。

当然，还有几个仓库没有转移的，因为组织无法创建永久期限的 GitHub 的 Token ，为了方便，还是暂时不做转移了。
